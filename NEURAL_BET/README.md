# NEURAL BET - The Double-Blind Oracle

## Mission
Système multi-agents de prédiction footballistique haute précision par "Reverse Engineering" (Contexte, Tactique, Variance).

## Architecture "Double Aveugle"
1. **Branche A : Le Labo** (Analyse Pure - sans cotes)
2. **Branche B : Le Marché** (Cotes)
3. **Synthesis** : Confrontation pour identifier la Value.

## Stack Technique
- **Langage** : Python 3.10+
- **Orchestration** : LangGraph / CrewAI (TBD)
- **LLM** : Claude 3 (Opus/Sonnet) / GPT-4o
- **Data** : Postgres (Structured), VectorDB (Context)
